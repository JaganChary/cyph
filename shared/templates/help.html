Cyph uses a version of Markdown called CommonMark for formatting.
We also support various extensions – including emojis and code fencing.

The following examples are shamelessly copied from reddit:

you type: you see:
*italics* italics
**bold** bold
[cyph me!](https://cyph.com) cyph me!

     if 1 * 2 < 3:
        print "hello!"
